https://dev.mysql.com/downloads/installer/ <br>

Developer Default（開発者向け）<br>
3. Check Requirements （要件を確認する）で下記をクリックする<br>
execute（実行する）<br>
4. Installation（取り付け）<br>
execute（実行する）<br>
